# PC Build Simulator

## 概要

初心者向けに自作PCの構成を簡単に作成し、見積書（PDF）として出力できるWebアプリ。

## 機能

- **パーツ選択**: CPU, GPU, メモリ, ストレージから選択
- **自動計算**: 選択したパーツの合計金額表示
- **構成コメント**: 構成に応じた自動コメント生成
- **PDF出力**: 見積書としてダウンロード可能

## 起動方法

```bash
# 仮想環境をアクティベート
source venv/bin/activate

# アプリを起動
streamlit run app.py
```

ブラウザで http://localhost:8501 にアクセスしてください。

## ファイル構造

```
pc-build-simulator/
├── app.py              # メインアプリケーション
├── parts.csv           # パーツデータ
├── requirements.txt    # 依存関係
└── venv/              # 仮想環境
```

## 技術スタック

- **Frontend**: Streamlit
- **Data**: CSVファイル
- **PDF生成**: ReportLab
- **言語**: Python 3.9+

## 企画書・要件定義書

### ■ アプリ名

PC Build Simulator（仮）

### ■ 背景・課題

* 自作PCは難しそうで敬遠されがち
* パーツ選びが複雑（相性・価格・用途）
* 初心者は「何を選べばいいか分からない」

### ■ 目的

* 自作PCのハードルを下げる
* 初心者でも簡単に構成を作れるようにする
* 構成を見積書として共有可能にする

### ■ ターゲット

* 自作PC初心者
* ゲーミングPCを検討しているライト層
* 友人に構成を相談したい人

### ■ 提供価値

* 直感的なUIでパーツ選択
* 自動で合計金額算出
* 見積書としてPDF出力
* 初心者向けの簡易コメント

### ■ MVP（最小構成）

* パーツ選択（CPU / GPU / メモリ / ストレージ）
* 合計金額表示
* PDF出力

### ■ 今後の拡張

* 相性チェック
* 電源容量計算
* AIによる構成提案
* データ自動更新（スクレイピング）

### ■ システム構成

* フロント＋バック：Streamlit（Python）
* データ：CSVファイル
* デプロイ：Streamlit Cloud

### ■ 機能要件

#### 1. パーツ選択機能

**内容**
ユーザーが各パーツを選択できる

**対象パーツ**
* CPU
* GPU
* メモリ
* ストレージ

**UI**
* セレクトボックス形式

#### 2. 見積表示機能

**内容**
選択したパーツを一覧表示

**表示項目**
* パーツ名
* 価格
* 合計金額

#### 3. 合計金額計算機能

**内容**
選択されたパーツの価格を合計

**条件**
* 未選択は0円扱い

#### 4. PDF出力機能

**内容**
見積内容をPDFとして生成・ダウンロード

**表示内容**
* タイトル（例：PC見積書）
* 構成一覧
* 合計金額
* 簡易コメント

#### 5. コメント生成（簡易）

**内容**
構成に応じてコメント表示

**例**
* GPU性能が高い → 「ゲーム向け構成」
* 低価格 → 「コスパ重視」

### ■ 非機能要件

**パフォーマンス**
* ページ表示は3秒以内

**可用性**
* 常時アクセス可能（Streamlit Cloud前提）

**保守性**
* パーツデータはCSVで管理
* コードとデータを分離

**拡張性**
* 将来的にAPI化可能な構造

### ■ データ設計

**ファイル名**
`parts.csv`

**カラム**
```
type,name,price,score,power
```

**説明**
* type：CPU / GPU / MEM / SSD
* name：商品名
* price：価格
* score：性能指標（任意）
* power：消費電力（任意）

### ■ 画面設計

**メイン画面**
* サイドバー：パーツ選択
* メイン：
  * 構成一覧
  * 合計金額
  * コメント
  * PDFダウンロードボタン

### ■ 技術要素

**使用ライブラリ**
* streamlit
* pandas
* fpdf または reportlab

### ■ 制約

* 外部APIは使用しない（初期）
* スクレイピングは行わない（初期）

### ■ 開発ステップ

**STEP1**
CSV作成

**STEP2**
UI作成（Streamlit）

**STEP3**
計算ロジック実装

**STEP4**
PDF出力機能実装

**STEP5**
デプロイ
